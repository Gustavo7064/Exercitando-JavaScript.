<h1>📌 Projeto - Gerador de Números Ímpares</h1>

<p>
Este projeto foi desenvolvido com o objetivo de praticar lógica de programação e manipulação com JavaScript.
</p>

<h2>🚀 Sobre o Projeto</h2>

<p>
A aplicação permite que o usuário digite um número qualquer e, a partir desse valor, 
o sistema retorna todos os números ímpares de 1 até o número informado.
</p>

<h2>💻 Tecnologias Utilizadas</h2>

<ul>
  <li>HTML5</li>
  <li>JavaScript</li>
</ul>

<h2>⚙️ Funcionamento</h2>

<p>
1. O usuário insere um número no campo de entrada.<br>
2. O sistema processa o valor utilizando JavaScript.<br>
3. São exibidos todos os números ímpares até o valor digitado.
</p>

<h2>📊 Exemplo</h2>

<p>
Entrada: <strong>900</strong><br>
Saída: <strong>1, 3, 5, 7, ..., 899</strong>
</p>

<h2>🎯 Objetivo</h2>

<p>
Este projeto foi criado com foco em aprendizado, reforçando conceitos como:
</p>

<ul>
  <li>Estruturas de repetição (loops)</li>
  <li>Condicionais</li>
  <li>Manipulação de DOM</li>
</ul>
